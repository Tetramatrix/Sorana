What is Sorana?

Sorana is a Visual AI File Organizer that transforms static folders into a living, visual environment. It's designed to tame digital chaos by combining spatial layouts with powerful no-code AI agent building capabilities.

Core Capabilities

  • Spatial AI Organization: Move beyond simple file lists. Sorana uses AI to semantically group your assets and arranges them on an intelligent 2D canvas, giving you an intuitive, bird's-eye view of your projects.
  • WYSIWYG Canvas Editor: Directly manipulate your workspace. Drag and drop files between groups, create new categories on the fly, rename items, and resize group boundaries with intuitive mouse controls. What you see is what you get.
  • Advanced Model Management: Connect to multiple cloud and local LLM backends including OpenAI, Mistral, Ollama, and Lemonade. The powerful model manager lists all available models and allows users to activate or deactivate models as needed for specific tasks.
  • No-Code Agent Orchestration: Build your own AI workforce visually. Create custom agents and connect them into intelligent pipelines using a simple drag-and-drop interface. AI: How It Works, Fine-Tuning & Manual Fallback

Why are my results "hit-or-miss"?

Sorana uses AI models to group and arrange icons. The default model is lightweight and works offline, but may not always classify every icon perfectly, especially for less common applications. Advanced users can connect larger or custom models for improved accuracy, but this may require more system resources.

  • Offline & Free (Default): The built-in model (~806MB) is downloaded on first run and works fully offline. It is fast, but may sometimes mislabel complex icons as "Miscellaneous."
  • Custom Models: For improved results, Sorana can connect to local services (like Ollama or Llama), OpenAI-compatible servers, or download larger models from Hugging Face.

Note: Larger models can provide better grouping, but require more memory and processing power.

Manual Fine-Tuning & Fallback

If the AI does not classify an icon as desired, or if no model is available, you can adjust the results manually using intiutive drag and drap and rename groups.
